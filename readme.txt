A simple jQuery UI Slider extension for contact form 7

== Description ==

This plugin comprises of jQuery UI Slider for contact form 7. If you have a number field who,s value you want to change with
the help of the slider, then you have come to the right spot.

Simple and elegant User Experience for your Contact Form users.

This is the long description.  No limit, and you can use Markdown (as well as in the following sections).



== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/cf7-ui-slider` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
